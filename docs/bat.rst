======================
Một vài BAT Command
======================

Đổi tên file:

.. code-block:: bash

    ren [tên file cần đổi] [tên file mới]

Tạo file mới:

.. code-block:: bash

    echo. >[tên file mói]

Liệt kê mỗi tên files trong directory, không hiển thị thêm thuộc tính nào khác:

.. code-block:: bash

    dir /b
